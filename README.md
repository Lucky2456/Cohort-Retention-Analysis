# WeCare | Customer Cohort & Retention Analysis (Power BI)

## Project Overview

This project analyzes customer acquisition, retention, repeat purchase behavior, revenue contribution, and category performance for **WeCare**, a beauty and personal care brand.

The analysis uses **Cohort Analysis** to track customer behavior over time and identify how customers acquired in one month continue purchasing in subsequent months.

Dashboard developed using **Power BI** with simulated transactional data generated using Python (Faker).

---

## Business Problem

Customer growth alone does not indicate business health.

This project answers:

* How many customers were acquired each month?
* How many customers returned in later months?
* Which acquisition channels generated higher revenue?
* Which product categories contributed most revenue?
* What percentage of customers became repeat buyers?

---

## Objective

Build an end-to-end cohort analysis dashboard to:

* Measure customer retention
* Track repeat purchase behavior
* Analyze revenue contribution across channels
* Understand category-level performance
* Generate actionable customer insights

## Cohort Methodology

### Cohort Definition

Customer Cohort =
Month of first purchase

Example:

Customer acquired in January remains part of the January cohort even if they purchase again in February or March.

### Cohort Index

Month 0 → Acquisition

Month 1 → Returned after 1 month

Month 2 → Returned after 2 months

* Qcom contributed highest revenue
